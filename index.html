B

Border radius

La propriété border radius permet d'arrondir les côtés d'un élément HTML. Il est également possible d'arrondir qu'un seul côté avec les propriétés border-top(ou bottom)-right(ou left)-radius.

Exemple
.rectangle-border-exemple{
    border-radius: 5px;
}

Il est également possible de faire des cercles avec le border-radius. 

.cercle {
    width: 100px; 
    height: 100px;
    border-radius: 50%;
    background-color: rgb(165, 199, 85);
    margin: 0 50px;
}

Box-shadow

La propriété box-shadow permet de créer une ombrage sur un élément. Il y a de 3 à 5 valeurs possible pour définir un box-shadow.

  • Offset-x | offset-y | color
  • Offset-x | offset-y | blur-radius | color
  • Offset-x | offset-y | blur-radius | spread-radius | color
Exemple
.rectangle-box-shadow {
    box-shadow: 5px 10px 5px #000000;
    margin-bottom: 20px;
}

Border-image

Permet d'avoir un image en bordure d'un élément HTML. La propriété border-image est le raccourcis des différentes propriétés de border-image:

  • border-image-source
  • border-image-slice
  • border-image-width
  • border-image-outset
  • border-image-repeat
Exemple avec une image png de cercle hachuré. 
.background-border-img{
    height: 100px;
    width: 100px;
    padding: 15px;
    border: 30px solid black;
    border-image-source: url(/img/—Pngtree—creative\ vector\ dotted\ line\ dotted_6030768.png);
    border-image-slice: 40%;
    border-image-repeat: repeat;
}

Box-sizing

Propriété qui définit si le navigateur va prendre en compte la bordure et le remplissage (padding) dans la valeur qui est définie pour la longueur et la largeur d’un élément. Content-box est la valeur de défaut, la taille total de l’élément est la somme de la largeur/hauteur définit et des bordures et remplissages alors que la valeur border-box ne va pas augmenter la taille de l’élément si on lui rajoute du padding ou une bordure puisque ces valeurs sont déjà prises en compte par le navigateur.

  • content-box (valeur par défaut)
  • border-box
Exemple avec la propriété content-box.
Exemple avec la propriété border-box. 
.box-sizing {
    box-sizing: content-box;
}
                    
.box-sizing-2 {
    box-sizing: border-box;
}

Filter

La propriété filter permet d'appliquer un filtre sur un élément. Il existe plusieurs filtres. Il est possible d'en utiliser plusieurs pour un seul et même élément.

  • blur
  • brightness
  • contrast
  • drop-shadow
  • grayscale
  • hue-rotate
  • invert
  • opacity
  • saturate
  • sepia
.chat-f1{
    filter: grayscale(100%); 
}
                    
.chat-f2 {
    filter:invert(100%);
}
                    
.chat-f3{
    filter:brightness(50%);
}

Text-decoration-line

Propriété qui permet de d'ajouter une ligne décorative selon les valeurs suivante: underline, overline, line-through et blink

  • blink
  • line-through
  • overline
  • underline

}

Text-shadow

La propriété text-shadow permet d'ajouter de l'ombre à un texte. Les valeurs qui peuvent être utilisé sont les même que pour box-shadow

Exemple

.text-shadow {
    text-shadow: rgb(6, 230, 43) 1px 0 10px;
    font-size: 2rem;
}

Tranform

La propriété transform permet de faire pivoter, de changer l'échelle (taille), de déplacer et d'incliner sur élément 2D ou 3D. Il est possible d'utiliser plusieurs valeurs pour un seul et même élément.

  • matrix
  • matrix3d
  • translate
  • translateX
  • translateY
  • translateZ
  • translate3d
  • scale
  • scaleX
  • scaleY
  • scaleZ
  • scale3d
  • skew
  • skewX
  • rotate
  • rotateX
  • rotateY
  • rotateZ
  • rotate3d
  • perspective

Parmi les formes géométriques suivantes, le parallélogramme (skew) et le diamant (rotate) ont été créés avec la propriété transform. Les autres formes ont été créées avec des borders (lelft, right et bottom). 

.parallelogram {
    width: 100px;
    height: 50px;
    background-color: rgb(165, 199, 85);
    transform: skew(20deg);
    margin: 20px;
    
}               

.diamant {
    width: 75px;
    height: 75px;
    background-color: rgb(165, 199, 85);
    transform: rotate(45deg);
    transform-origin: 24% 96%;
    margin-right: 30px;
}                   

.triangle {
    width: 0;
    height: 0;
    border-left: 40px solid transparent;
    border-right: 40px solid transparent;
    border-bottom: 80px solid rgb(165, 199, 85);
}                

.triangle-rectangle{
    width: 0;
    height: 0;
    border-bottom: 75px solid rgb(165, 199, 85);
    border-right: 75px solid transparent;
}             

.trapeze {
    border-bottom: 50px solid rgb(165, 199, 85);
    border-left: 25px solid transparent;
    border-right: 25px solid transparent;
    height: 0;
    width: 50px;
    margin-top: 50px;
}

Transition

Propriété qui est en fait le raccourci pour les propriétés :

  • transition-proprety
  • transition-duration
  • transition-timing-function
  • transition-delay

Elle permet de définir la transition entre deux états d’un élément.

Survole-moi 
.rectangle-transition:hover {
    color: white;
    background-color: rgb(5, 51, 11);
    transition: 1s ease-in-out;
    }

  • vertical-rl

Exemple

    writing-mode: vertical-lr;
}
